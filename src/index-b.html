I²C lets you vastly extend the Pi's capabilities, by connecting a variety of other devices. Examples include temperature sensors, LCD displays and Analogue to Digital Converters.

The first tutorial, Using I2C On The Raspberry Pi, explains how to configure the hardware and software of the Pi to get I²C up and running. If you already know what you want to do with I²C, then this is an ideal place to start.

You may find that some or all of the devices you want to connect use a different voltage for communication than the Pi (so called logic levels.) In this case, first build a bidirectional level shifter to convert the voltages, which is described in the second tutorial.

If you're stuck for ideas, then why not read about using I²C to interconnect Arduinos with the Raspberry Pi? This gives you the variety of low-level I/O options and dedicated processing of a microcontroller, coupled with all the flexibility and processing speed of the Raspberry Pi.

Not only can you communicate with other devices using I²C, it's also possible to turn them on and off with a simple circuit, described in the fourth tutorial. For example, this could be used to turn devices off when not in use, saving power.
